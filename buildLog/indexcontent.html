Software Controlled Drone - Build Instructions

Introduction

The Software Controlled Drone project will allow users to control a Syma X12 Nano QuadCopter Drone with an Android application, which will use an Arduino and a nRF24L01 transciever to communicate with the drone.

Step 2: Connect the nRF24L01 Transceiver to the Arduino

This is the pin set up from the Arduino to the nRF24L01 Transceiver:

5v -> VCC
GND -> GND
D8 -> IRQ
D9 -> CE
D10 -> CSN
D11 -> MOSI
D12 -> MISO
D13 -> SCK

Power Up

Once the circuit has been built, plug the Arduino into the PC and open the Arduino IDE. Import the library into the Arduino IDE by going to Sketch->Include Library->Add .ZIP Library. Once the library has been added into the IDE, copy the code from here. Once the code is copied, upload it by clicking the arrow icon on the top left of the IDE. When it has finished uploading, open the serial monitor by going to Tools->Serial Monitor. You are now able to communicate with another transceiver.
NOTE: enable for two transceivers to communicate with each other, one user will have to have a radioNumber = 0 and another user to have a radioNumber = 1. This is changed inside the arduino code.

Is This Project Reproducible?

This project is reproducible if the given build instructions are followed and if the wiring of the circuit is done correctly.
