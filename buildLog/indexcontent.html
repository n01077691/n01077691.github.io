Power Up

Start your Arduino IDE right after the circuit is built. Import the library into the Arduino IDE (Sketch->Include Library->Add .ZIP Library -> Select RF24.zip). Once the library has been added into the IDE, get the code from repository. To establish communication between two transceivers one user will have to set radioNumber = 0 and another user has to change to radioNumber = 1. Once the code is compiled, upload it by clicking the arrow icon on the top left of the IDE. Next, check for any compilation errors and open the serial monitor (Tools->Serial Monitor). You will now be able to communicate with another sparkFun transceiver.
Be sure that LED of the Sender goes RED when transceiving times out and GREEN when data is successfully transmitted. LED of the receiver will be always GREEN, since it is always trying to receive any data. On the Serial Monitor you should be able to see all details about packets sent.
