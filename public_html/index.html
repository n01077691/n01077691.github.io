Game: Battleship

Click on the title or the picture to start the game!


Basic javascript implementation of the game Battleship. Three ships are randomly placed on the board, either vertically or horizontally. The ships do not overlap. At the end the game will show your accuracy. Good Luck!

Game: Order vs Chaos

Simple example of board game Order and Chaos where two players try to achieve order in figures or chaos on board. There are several versions of this game.


English version: Order_vs_Chaos(English)
